# Kylin Knowledge建立目的

Kylin Knowledge的建立主要为了解决以下问题：

* 以一定层次展示个人知识储备和项目
* 发布`LeetCode` `Project Euler` `Hackerrank` 等网站的习题解答
* 为计算机知识学习感到迷茫和困惑的人提供一个参考
